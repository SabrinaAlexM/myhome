# 🏡 Mein Zuhause

Persönliche App für Haushalt & Me Time. Gebaut mit React + Vite.

## Features

- **🏠 Haushalt** – Putzaufgaben nach Raum, Intervalle, Fortschrittsbalken
- **🌸 Me Time** – Persönliche Termine (Coiffeur, Maniküre etc.) mit Min/Max-Intervall und smarten Terminvorschlägen
- **Onboarding** – Name, Beispieldaten
- **Einstellungen** – Name ändern, Daten zurücksetzen
- Daten lokal im Browser (localStorage)

## Setup

```bash
npm install
npm run dev
```

## Deploy auf Vercel

1. Repo auf GitHub pushen
2. Auf [vercel.com](https://vercel.com) mit GitHub verbinden
3. Projekt importieren → **Vite** wird automatisch erkannt
4. Deploy klicken ✓
